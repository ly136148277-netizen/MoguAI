          Agent 介绍

          Agent 是什么？

          Agent 是 MOGU AI 的「办事助手」：侧栏点 Agent，打字即可让本机办事，或问怎么用。日常出片请走创作台。

          建议先在「环境」配好 Ollama / PAI / ComfyUI，再（可选）到「模型」给 Agent 配引导脑子。那是答疑用的模型，不是 ComfyUI 出片模型。

          常用能力（记住这几项就够）

          • 打开软件 — 如 ComfyUI、Ollama、资源管理器
          • 列出 / 分析工作流 — 看本机有哪些出片流
          • 确认出片 — 一句话跑工作流（可视化出片仍用创作台）
          • 搜索文件 — 全文搜本机索引（如「搜索 ROADMAP」）
          • 备份 — 打包 PAI / 项目 zip
          • 恢复 — 从备份还原
          • 抓取网页 — 抓页面，可用本地模型总结
          • 删除文件 — 需二次确认；删前自动备份快照
          高级能力（完整控制台，多数人用不上）
          • 启动时开始 — 一键自检并拉起 Ollama/ComfyUI、建索引（日常看「环境」页灯即可）
          • 视频后期 — ffmpeg 剪辑/拼接（一般更建议剪映）
          • 更新 / 管理 ComfyUI — 运维向，有误操作风险
          • 索引目录、下载链接、软件更新、定时任务、批量操作 — 引擎支持，不在此页展开

          怎么开始

          1. 首页确认环境灯为绿，或到「环境」补齐
          2. （可选）「模型」→ 本地/联网，给 Agent 配引导脑子
          3. 侧栏进入 Agent，打字或点快捷按钮

            这是什么？
            
          
            MOGU AI 帮助你在电脑上下载、管理并使用 AI 大语言模型，并集成 AI 执行管家完成 ComfyUI 出片、搜文件等任务。
            
          
            需要什么？
            
          
            使用 AI 聊天前，请先安装并启动 Ollama。顶部显示「本地 AI 引擎已就绪」即可聊天；若显示「未运行」，请点击 启动 Ollama。
            
          
            完整流程
            
          
              
            
            1. 打开「模型仓库」，找到想要的模型，点击「下载」
              2. 
            
            2. 在「下载中心」查看进度，完成后会自动导入
              3. 
            
            3. 打开「聊天」，选择已导入的模型开始对话
              4. 
          
            
          
            Agent 怎么用
            
          
              
            
            • Agent 对话窗：直接打字办事（打开 ComfyUI、列工作流、搜文件、备份、删文件等）或问「怎么用创作台」。
              • 
            
            • 模型闲聊：Agent 页右上角入口，用 Ollama 本地模型写文案、闲聊（需先下载并导入模型）。
              • 
            
            • 创作台：挂工作流出片；也可从 Agent 点「去创作台」。
              • 
          
            
          
            常见问题
            
          
            下载慢？可在「设置」中切换下载线路或开启「启动时自动启动 Ollama」。聊天/导入失败？请确认 Ollama 已启动。改保存位置后，已下载文件不会自动移动，仅新下载存到新目录。
