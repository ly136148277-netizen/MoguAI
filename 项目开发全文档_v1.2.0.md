## 二、项目是什么

**AI 模型管理助手**（AI Model Manager）是一款 **Electron 桌面应用**，面向普通用户，实现：

```
浏览模型 → 下载 GGUF → 自动导入 Ollama → 本地离线聊天
```

- **技术栈：** Electron 35 + Node.js + 原生 HTML/CSS/JS（无 React/Vue）
- **外部依赖：** 本机需安装并运行 [Ollama](https://ollama.com/)
- **当前版本号：** `1.2.0`（见 `package.json`）

## 四、系统架构（三层 + IPC）

```
┌─────────────────────────────────────────────────────────────┐
│                    渲染进程 (Renderer)                       │
│  index.html → app.js(AppRouter) → page-controller.js       │
│  renderer.js(AppCore) │ chat.js │ pages/my-models-page.js │
└──────────────────────────┬──────────────────────────────────┘
                           │ window.modelManager
┌──────────────────────────▼──────────────────────────────────┐
│                    预加载 (preload.js)                       │
│              contextBridge + ipcRenderer                     │
└──────────────────────────┬──────────────────────────────────┘
                           │ IPC invoke / events
┌──────────────────────────▼──────────────────────────────────┐
│                    主进程 (main.js)                          │
│  ModelRepository │ DownloadEngine │ OllamaService            │
│  StorageManager  │ SettingsStore  │ ChatSessionStore         │
└──────────────────────────┬──────────────────────────────────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
         本地文件系统    Ollama CLI/API   HuggingFace 等
```

**安全模型：**
- `contextIsolation: true`（渲染进程无法直接访问 Node）
- `nodeIntegration: false`
- 所有系统能力通过 `preload.js` 暴露的 `window.modelManager` 调用

## 七、核心代码说明（带文件位置）

### 7.1 页面路由 — `src/renderer/app.js`

```javascript
function navigate(page) {
  document.querySelectorAll(".view").forEach((v) => v.classList.remove("is-active"));
  document.getElementById(`view-${page}`).classList.add("is-active");
  currentPage = page;
  handlers[page]?.();  // 进入页面时加载数据
}
```

### 7.2 页面注册 — `src/renderer/page-controller.js`

```javascript
window.AppRouter.onPage("home",     () => window.AppCore.loadDashboard());
window.AppRouter.onPage("models",   () => window.AppCore.loadModels());
window.AppRouter.onPage("downloads",() => window.AppCore.loadQueue());
window.AppRouter.onPage("my-models",() => window.MyModelsPage.load());
window.AppRouter.onPage("settings", () => window.AppCore.loadSettingsForm());
window.AppRouter.onPage("help",     () => {});
```

### 7.3 从模型页进入聊天 — `src/renderer/chat.js`

```javascript
function enterWithModel(model) {
  if (!model?.ollamaImported) return false;
  pendingModel = model;
  window.AppRouter.navigate("chat");
  return true;
}
```

### 7.4 删除模型 IPC — `src/main/main.js` + `src/preload/preload.js`

主进程处理 `models:delete`：删除 GGUF 文件、Modelfile、Ollama 模型。  
渲染进程通过 `window.modelManager.deleteModel(modelId)` 调用。

### 7.5 隐藏菜单栏 — `src/main/main.js`

```javascript
// 创建窗口时

// app ready 后
Menu.setApplicationMenu(null);
```

## 八、IPC 接口一览

渲染进程通过 `window.modelManager.*` 调用，定义在 `src/preload/preload.js`。

| 分类 | 方法/事件 | 作用 |
|------|-----------|------|
| 模型 | `listModels()` | 获取模型列表 |
| 模型 | `syncCatalog()` | 同步远程目录 |
| 模型 | `toggleFavorite()` | 收藏切换 |
| 模型 | `deleteModel()` | **v1.2 新增** 删除模型 |
| 下载 | `startDownload()` / `pauseDownload()` 等 | 下载控制 |
| 下载 | `onDownloadProgress()` | 进度事件 |
| Ollama | `importModel()` / `removeModel()` | 导入/移除 |
| Ollama | `onOllamaImportComplete()` | 导入完成事件 |
| 聊天 | `sendChat()` / `onChatChunk()` | 流式对话 |
| 设置 | `getSettings()` / `updateSettings()` | 读写设置 |
| 应用 | `getVersion()` / `getDashboardStats()` | 版本与统计 |

完整列表见 `PROJECT_CONTEXT.md` 第六节。
