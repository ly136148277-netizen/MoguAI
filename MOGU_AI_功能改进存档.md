## 四、本轮已落地（2026-07-19，供对照）

- 精密工厂：JS/TS 补全/跳转、Node 调试
- 编程引擎：`config/moguai-runtime-compat.json` + 设置「检查更新 / 安装升级」
- 策略：跟官方版本、应用钉扎适配、不盲升未适配 latest
- **开箱缩短**：对话/工厂出现「安装编程引擎」CTA；设置进页自动检查；失败文案改为一键安装
- **派工闭环**：打开工厂带 prompt；任务卡「再派工」；任务 `logSummary` 含改动摘要；大脑回复带 review
- **干活能力续**：按文件接受/拒绝、全部接受/拒绝；工厂运行/测试输出面板；派工取消；失败带日志尾与续派提示
- **编程收尾加强**：引擎 B `uv` 依赖重试；调用栈 + 条件断点；工厂文件名/符号搜索
- **旧阶段“一个窗口 + 基础干活能力”已完成**；不等于 Frontier Coding Agent 完成，后续按 2.1/2.2 推进。
- 测试基线：见当时 `npm test`
