## Capability decisions

### Repo intelligence, LSP and test discovery

- Benefit observed: real repository indexed 241 code files, 3,483 symbols and 5,921 conservative
  call edges; target symbol returned one definition and thirteen references; 68 tests were discovered.
- Cost: index memory and refresh CPU; static calls remain conservative and language-dependent.
- Security: read-only roots and bounded files; LSP processes have controlled cwd/env/lifecycle.
- License: MOGU internal MIT implementation; no language server is bundled.
- Decision: retain Default-Off pending model A/B.

### Controlled terminal and managed worktrees

- Benefit observed: real ConPTY command exited 0 with start/finish audit; a detached exploration
  worktree was created at the exact baseline and removed cleanly.
- Cost: native ABI/build maintenance and temporary disk use.
- Security: terminal is always L3-authorized; root/env/time/output/concurrency limits apply.
  Worktrees are manager-owned and capability-read-only, but not an OS read-only mount.
- License: `node-pty@1.1.0` and `node-addon-api@7.1.1`, both MIT, exact lock/SRI/Notice audited.
- Decision: retain Default-Off; require signed packaged native-module E2E before release use.

### Events, leases, retry, subtasks and recovery

- Benefit observed: append/replay and redaction passed; a one-use run-bound lease consumed exactly
  once then denied; local retry and two-subtask checkpoint paths pass deterministic tests.
- Cost: JSONL growth and single-process serialization; no global scheduler.
- Security: append failures stop side effects; L3 cannot be leased; Gateway-accepted tasks are never
  automatically resubmitted; read-only subtasks cannot write/commit/push/install.
- License: MOGU internal MIT implementation.
- Decision: retain Default-Off. Full DAG, migration and global budget remain 2.3 work.

### GPT-5.6 Brain Adapter

- Benefit observed: contract behavior passes no-fallback, key isolation, normalized tool/usage/trace,
  timeout, abort, size and stable secret-free configuration-hash tests.
- Cost: provider pricing/budget enforcement and real latency are not yet measured.
- Security: key resolves only at request time; endpoint/model/config are explicit; unavailable
  configuration returns typed `BLOCKED`.
- License: MOGU internal OpenAI-compatible protocol implementation; no provider SDK imported.
- Decision: retain Default-Off.
