## Capability decisions

### Explainable routing and budget ledger

- Benefit observed: deterministic classification, owner-ordered profile selection, typed `BLOCKED`
  without silent model substitution, and idempotent run/day reservations.
- Cost: owner must maintain model profiles and task policies; unknown pricing cannot satisfy
  cost-denominated budgets.
- Security: secret values forbidden in settings profiles; credential presence is reference-only.
- License: MOGU clean-room MIT; no new npm.
- Decision: retain Default-Off pending model A/B.

### Neural planner and pinned LSP

- Benefit observed: real-repo NeuralPlan composes scope, static index evidence, test discovery and
  verify stages; missing/invalid LSP registration falls back to static evidence.
- Cost: index/LSP wall time inside planner budgets; external servers remain operator-owned.
- Security: path escape fail-closed; LSP requires `version` + `licenseEvidenceId`; no auto download.
- License: client is MOGU MIT; each concrete server needs its own License evidence before recommend.
- Decision: retain Default-Off.

### Context budget, tool-chain and decision trace

- Benefit observed: secret-key redaction, hard byte/token caps, phase allowlists that never expand
  the advertised tool set, and replayable decision traces.
- Cost: aggressive eviction may drop optional evidence; phase recoveries are capped.
- Security: investigate cannot apply patches; Gateway-accepted tasks stay outside closed-loop replay.
- License: MOGU clean-room MIT.
- Decision: retain Default-Off.

### Bounded closed loop

- Benefit observed: in-task checkpoint → execute → verify → replan → retry → review within
  `maxRepairIterations` and other budgets.
- Cost: repair iterations consume wall time/steps/tool calls; replan failure is typed blocked.
- Security: never auto-resubmits Gateway-accepted work; permission denial is terminal.
- License: MOGU clean-room MIT wrapping the existing coding loop.
- Decision: retain Default-Off.
